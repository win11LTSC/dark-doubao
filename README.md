# dark-doubao
将豆包网页版改为暗黑的油猴脚本
[点击安装（较慢）](https://github.com/win11LTSC/dark-doubao/raw/refs/heads/main/%E8%B1%86%E5%8C%85%E7%BD%91%E9%A1%B5%E7%89%88%E9%BB%91%E6%9A%97%E4%B8%BB%E9%A2%98-1.0.user.js)或[国内加速安装（较快）](	https://gh.xmly.dev/https://github.com/win11LTSC/dark-doubao/raw/refs/heads/main/%E8%B1%86%E5%8C%85%E7%BD%91%E9%A1%B5%E7%89%88%E9%BB%91%E6%9A%97%E4%B8%BB%E9%A2%98-1.0.user.js)
若不想将豆包网页变暗黑关掉此脚本即可
注意需安装篡改猴或暴力猴的插件
